# Template

This file exists at `content/writing/_template.mdx` solely so that Turbopack can
resolve dynamic imports of the form ``await import(`@/../content/writing/${slug}.mdx`)``
in `src/app/writing/[slug]/page.tsx`. It is intentionally not listed in
`internalPosts` in `src/lib/writing.ts`, so it has no `/writing/_template`
route and is never reachable through the site.

Leave this file in place unless you also rewrite the dynamic-import pattern in
the post route to a static map of slug → MDX module.
